# create-uno

> Lightweight generator for frontend, backend, and DevOps templates.

[![npm version](https://img.shields.io/npm/v/create-uno.svg)](https://www.npmjs.com/package/create-uno)
[![node version](https://img.shields.io/node/v/create-uno.svg)](https://nodejs.org)
[![license](https://img.shields.io/npm/l/create-uno.svg)](https://github.com/keyfunc/uno/blob/master/LICENSE)

## Usage

```bash
pnpm create uno
```

Or:

```bash
npm create uno@latest
yarn create uno
```

## Templates

Templates are loaded from [templates.json](https://github.com/keyfunc/template/blob/master/templates.json).

## Requirements

- Node.js >= 18.0.0
