# Changelog

## [1.0.1]

### Changed
- Add package dependencies required by the `uno` CLI.
- Add development dependencies required for local builds.
- Add repository, issue tracker, and homepage metadata.
- Update the npm publish workflow to install, build, dry-run pack, and publish with provenance.

## [1.0.0]

### Added
- Initial release of `@keyfun/uno`.
- Add the `uno` CLI entry.
- Add `uno generate component <name>` for React component templates.
- Add `uno generate store <name>` for Zustand store templates.
- Add `uno generate service <name>` for service templates.
- Support the `uno g` command alias and short aliases for component, store, and service generation.
