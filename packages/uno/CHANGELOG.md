# Changelog

## [1.0.0]

### Added
- Initial release of `@keyfunc/uno`.
- Add the `uno` CLI entry.
- Add `uno generate component <name>` for React component templates.
- Add `uno generate store <name>` for Zustand store templates.
- Add `uno generate service <name>` for service templates.
- Support the `uno g` command alias and short aliases for component, store, and service generation.
